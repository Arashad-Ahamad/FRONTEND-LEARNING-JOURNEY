# 🛍️ ShopHub - E-Commerce Web App

A modern and fully responsive e-commerce web application built using **React, Redux Toolkit, and Tailwind CSS**.
This project demonstrates real-world features like cart management, wishlist, product filtering, and API integration.

## 📌 Features

### Core Features
- 🏠 **Clean & Responsive Home Page** - Hero section, category tabs, featured products
- 🛒 **Cart Management** - Add/remove items, update quantity, persistent storage
- ❤️ **Wishlist** - Save favorite products for later
- 🔍 **Product Search** - Real-time search with debouncing
- 📂 **Category Filtering** - Filter products by category
- 💰 **Price Filter** - Filter products by price range
- 📦 **Product Detail Page** - Dynamic product information
- 💳 **Checkout Flow** - Shipping & payment form

### Technical Features
- ⚡ **API Integration** - FakeStore API for product data
- 💾 **LocalStorage Persistence** - Cart & wishlist survive page refresh
- 🔔 **Toast Notifications** - User-friendly feedback
- 📱 **Fully Responsive** - Works on mobile, tablet, and desktop
- 🎨 **Modern UI** - Clean, professional color scheme
