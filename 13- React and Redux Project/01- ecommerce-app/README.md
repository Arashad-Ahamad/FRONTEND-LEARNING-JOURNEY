# 🛍️ ShopHub - E-Commerce App

Modern e-commerce web app built with React, Redux Toolkit, and Tailwind CSS.

## 🚀 Tech Stack

- React 18 + Vite
- Redux Toolkit (State Management)
- React Router DOM v6
- Tailwind CSS v4
- Axios (API calls)
- React Hot Toast (Notifications)
- FakeStore API

## ✨ Features

### Cart Features
- ✅ Add/Remove items
- ✅ Update quantity (+/-)
- ✅ Auto-calculate total price + shipping + tax
- ✅ LocalStorage persistence (cart saves on refresh)
- ✅ Clear cart option
- ✅ Toast notifications

### Product Features
- ✅ Product listing grid (responsive)
- ✅ Product detail page
- ✅ Product images with hover zoom
- ✅ Rating stars display

### UI/UX Features
- ✅ Loading spinner
- ✅ Error handling with retry button
- ✅ Sticky navbar with cart badge
- ✅ Footer with social links
- ✅ Hero section with CTA button
