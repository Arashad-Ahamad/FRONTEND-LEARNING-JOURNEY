# 📝 React Todo App

A simple and clean Todo App built using React.js.
This app allows users to add, delete, check, and store tasks using localStorage.

## 🚀 Features

* Add new tasks
* Prevent duplicate tasks
* Mark tasks as completed ✅
* Delete individual tasks ❌
* Clear all tasks 🧹
* Live date & time display ⏰
* Data saved in localStorage (no data loss on refresh)

## 🛠️ Tech Stack

* React JS (useState, useEffect)
* JavaScript (ES6)
* CSS3
* LocalStorage API
