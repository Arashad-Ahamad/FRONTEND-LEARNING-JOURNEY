Decision Making in JavaScript using If Statement

In JavaScript, the if statement is used for decision-making based on a condition. If the condition is true, the code inside the if block will execute. If the condition is false, the code will not execute.

Step 1: Basic Syntax of if Statement

    if (condition) {
        // This code runs only if the condition is true
    }

If the condition is true, the inner code executes. Otherwise, nothing happens.

Step 2: Simple Example

Check if a number is greater than 10:

    let number = 15;

    if (number > 10) {
        console.log("Number is greater");
    }

Explanation:

  • number = 15, and 15 > 10 is true, so "Number is greater" will be printed.
  • If number = 5, then 5 > 10 is false, so nothing will be printed.

Output:

  • number = 15: "Number is greater"

Step 3: if-else Statement (Two Conditions)

Use if-else when there are two possible conditions: either the condition is true or false.

That means if the condition is true, the if block will execute, and if the condition is false, the else block will execute. 

    let age = 17;

    if (age >= 18) {
        console.log("You can vote");
    } else {
        console.log("You cannot vote");
    }

Explanation:

  • If age = 19, then 19 >= 18 is true, so "You can vote" will be printed.
  • If age = 17, then 17 >= 18 is false, so the else block will execute and "You cannot vote" will be printed.

Output:

When age = 19: "You can vote"

When age = 17: "You cannot vote"

Step 4: if-else if-else (Multiple Conditions)

Use else if when multiple conditions need to be checked:

    let marks = 80;

    if (marks >= 90) {
        console.log("Grade A");
    } else if (marks >= 75) {
        console.log("Grade B");
    } else if (marks >= 50) {
        console.log("Grade C");
    } else {
        console.log("Grade F");
    }

Explanation:

  • If marks = 80, then 80 >= 90 is false, so this condition will be skipped.
  • Then, 80 >= 75 is true, so "Grade B" will be printed.
  • The remaining conditions will not be checked because once a correct condition is found, the rest are ignored.

Output:

  • marks = 80: "Grade B"
  • marks = 45: "Grade F" (because 45 is less than 50, so the else block will execute)

Summary:

  • if – The first condition is checked. If it is true, the code runs, and the rest of the conditions are not checked.
  • else if – If the first if is false, then the next condition is checked. If this condition is true, the code runs, and the rest of the conditions are not checked.
  • else – If both if and else if are false, then the else block runs.

Step 5: Nested if Statement

When one if is inside another if, it is called a nested if statement: 

    let num = 12;

    if (num > 0) {
        if (num % 2 === 0) {
            console.log("This is a positive and even number");
        } else {
            console.log("This is a positive but odd number");
        }
    }

Explanation:

  • First, it will check whether num > 0. If true, the inner if statement will be checked.
  • Then, it will check whether num % 2 === 0.
  • If true, it will print "This is a positive and even number".
  • If false, the else block will execute and print "This is a positive but odd number".

Output:

  • num = 12: "This is a positive and even number"
  • num = 13: "This is a positive but odd number"

Step 6: Where to Use If Statements?

  • Checking if a number is positive or negative
  • Checking a user's age
  • Assigning grades based on marks
  • Controlling events in a program

Summary

  • if – Executes when the condition is true.
  • if-else – Executes one block for true and another for false.
  • if-else if-else – Checks multiple conditions.
  • Nested if – An if inside another if.
