Comparison Operators in JavaScript

Comparison operators in JavaScript are used to compare values. These operators return a boolean result (true or false) based on the comparison.

  • == → Equal to (Compares values but ignores the type)
    Example: 5 == '5' → true (values are the same, type is ignored)

  • === → Strictly equal to (Compares both value and type)
    Example: 5 === '5' → false (value is the same, but type is different)
    Note: Returns true only if both value and type are the same.

  • != → Not equal to (Compares values but ignores the type)
    Example: 5 != '5' → false (values are the same)

  • !== → Strictly not equal to (Compares both value and type)
    Example: 5 !== '5' → true (value is the same, but type is different)
    Note: Returns true if either the value or the type is different.

  • > → Greater than (Checks if the left value is greater than the right value)
    Example: 10 > 5 → true

  • < → Less than (Checks if the left value is less than the right value)
    Example: 5 < 10 → true

  • >= → Greater than or equal to (Checks if the left value is greater than or equal to the right value)
    Example: 10 >= 10 → true

  • <= → Less than or equal to (Checks if the left value is less than or equal to the right value)
    Example: 5 <= 10 → true
