Truthy and Falsy Values in JavaScript

In JavaScript, any value can be converted to `true` or `false` using the `Boolean()` function. Values that return `true` are called **Truthy** values, and values that return `false` are called **Falsy** values.

Truthy Values:

Values that return `true` when passed to the `Boolean()` function are considered **Truthy**.

  • Non-empty strings → '0', 'false', 'hello', etc.
  • Non-zero numbers → 42, -42, 3.14,10,20 etc.
  • Objects → {}, {name: "Arashad"}
  • Arrays → [], [1, 2, 3]
  • Functions → function(){}, ()=>{}
  • Special values → Infinity, -Infinity

Falsy Values:

Values that return `false` when passed to the `Boolean()` function are considered **Falsy**.

  • 0 → The number zero
  • -0 → Negative zero
  • NaN → Not a Number
  • '' → Empty string
  • undefined → A value that is not defined
  • null → A null value
