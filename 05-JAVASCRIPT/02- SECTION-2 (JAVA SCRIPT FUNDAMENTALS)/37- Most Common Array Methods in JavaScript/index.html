JavaScript Array Methods

Add/Insert Methods

  • push() – Add at end
  • unshift() – Add at beginning
  • splice() – Insert at specific index
  • concat() – Joining two or more arrays

Remove Methods

  • pop() – Remove from end
  • shift() – Remove from beginning
  • splice() – Remove at specific index

Access / Info

  • indexOf() – Index of value
  • includes() – Checks if value exists

Copy / Extract

  • slice() – Copy part of array

Rearrange / Sort

  • reverse() – Reverse order of array
  • sort() – Sort array

Modify / Change

  • splice() – Add/Remove/Change items

Properties

  • length – Size of array
