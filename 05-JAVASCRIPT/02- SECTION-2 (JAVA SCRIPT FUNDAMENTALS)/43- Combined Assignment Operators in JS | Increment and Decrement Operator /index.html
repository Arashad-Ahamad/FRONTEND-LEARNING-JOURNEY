Compound (Combined Assignment) Operators

Compound operators are used to update the value of a variable using an operation. Below are the most common ones:

  • += : Addition Assignment Operators
    → It adds the given value and updates the variable.
    Example: x += 5 is same as x = x + 5
  • -= : Subtraction Assignment Operators
    → It subtracts the given value and updates the variable.
    Example: x -= 3 is same as x = x - 3
  • *= : Multiplication Assignment Operators
    → It multiplies the given value and updates the variable.
    Example: x *= 2 is same as x = x * 2
  • /= : Division Assignment Operators
    → It divides the variable by the given value and updates it.
    Example: x /= 4 is same as x = x / 4
  • %= : Remainder Assignment Operators
    → It gets the remainder and updates the variable.
    Example: x %= 2 is same as x = x % 2
  • **= : Exponentiation Assignment Operators
    → It raises the variable to the power and updates it.
    Example: x **= 3 is same as x = x ** 3

Increment and Decrement Operators

  • ++ : Increment Operator
    → It increases the value by 1.
    Example: x++ means x = x + 1
  • -- : Decrement Operator
    → It decreases the value by 1.
    Example: x-- means x = x - 1
