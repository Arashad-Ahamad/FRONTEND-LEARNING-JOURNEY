Logical Operators in JavaScript

Logical operators are used to combine multiple conditions and return true or false based on the given conditions.

  1. && (AND) Operator

    Returns true only if both conditions are true.

    If any one condition is false, the result is false.

    Example 1 (True Case):

    let age = 20;
let hasTicket = true;
console.log(age >= 18 && hasTicket); // true

    Explanation:
    - age >= 18 → (True)
    - hasTicket → (True)
    - True && True → Result: True

    Example 2 (False Case):

    let age = 16;
let hasTicket = true;
console.log(age >= 18 && hasTicket); // false

    Explanation:
    - age >= 18 → (False)
    - hasTicket → (True)
    - False && True → Result: False

  2. || (OR) Operator

    Returns true if at least one condition is true.

    False only if both conditions are false.

    Example 1 (True Case):

    let hasID = false;
let hasPassport = true;
console.log(hasID || hasPassport); // true

    Explanation:
    - hasID → (False)
    - hasPassport → (True)
    - False || True → Result: True

    Example 2 (False Case):

    let hasID = false;
let hasPassport = false;
console.log(hasID || hasPassport); // false

    Explanation:
    - hasID → (False)
    - hasPassport → (False)
    - False || False → Result: False

  3. ! (NOT) Operator

    Reverses the boolean value (True becomes False and False becomes True).

    Example 1 (True Case):

    let isOnline = false;
console.log(!isOnline); // true

    Explanation:
    - isOnline → (False)
    - !False → Result: True

    Example 2 (False Case):

    let isOnline = true;
console.log(!isOnline); // false

    Explanation:
    - isOnline → (True)
    - !True → Result: False

Usage of Logical Operators

Logical operators are used when we need to combine multiple conditions to make a decision.
