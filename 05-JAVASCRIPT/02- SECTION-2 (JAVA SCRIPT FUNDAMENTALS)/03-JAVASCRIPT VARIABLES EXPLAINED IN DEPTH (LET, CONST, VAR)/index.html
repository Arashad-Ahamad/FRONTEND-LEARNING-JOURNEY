Variables in JavaScript

What is a Variable?

A variable is a name given to a storage location in JavaScript.

Ways to Declare a Variable:

In JavaScript, we can declare a variable using three keywords:

  • let - Used for block-scoped variables.
  • const - Used for constants that cannot be reassigned.
  • var - An older way to declare variables (function-scoped).

Rules for Naming Variables:

While naming a variable, follow these rules:

  • Letters: Can use uppercase (A-Z) and lowercase (a-z) letters.
  • Digits: Can use numbers (0-9), but the variable name cannot start with a digit.
  • Special Characters: Only $ (dollar sign) and _ (underscore) are allowed.
  • Case-sensitive: JavaScript variables are case-sensitive (e.g., myVar and myvar are different).
  • No Reserved Words: Cannot use JavaScript reserved keywords as variable names (e.g., let, const, function).
