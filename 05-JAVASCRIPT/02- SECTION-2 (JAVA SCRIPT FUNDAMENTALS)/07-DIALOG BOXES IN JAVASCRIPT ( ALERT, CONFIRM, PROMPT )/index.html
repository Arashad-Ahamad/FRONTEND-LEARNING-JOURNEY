Dialog Boxes

There are three types of Dialog Boxes

  1. alert("Your message goes here.") → No return value → undefined ✔ Used to show a message or notification. Example: 
    alert("Welcome to my website!");
// Output: Shows a popup message with "Welcome to my website!"
  2. confirm("Your message goes here.") → Returns true or false ✔ Used to get confirmation from the user. Example: 
    const response = confirm("Are you sure?");
console.log(response);
// Output: true (if OK is clicked) or false (if Cancel is clicked)
  3. prompt("Your message goes here.") → Returns user input or null ✔ Used to take input from the user. Example: 
    const name = prompt("What is your name?");
console.log(name);
// Output: User input or null (if Cancel is clicked)
