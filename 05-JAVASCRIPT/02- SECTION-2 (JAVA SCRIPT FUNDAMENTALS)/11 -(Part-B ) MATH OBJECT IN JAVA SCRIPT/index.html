Math Object in JavaScript

Properties on Math Object

  • Math.PI → Returns the value of π (3.141592653589793)
  • Math.SQRT2 → Returns the square root of 2 (1.414)
  • Math.E → Returns Euler's constant (2.718)

Methods on Math Object

  • Math.sqrt(x) → Returns the square root of x (Math.sqrt(9) = 3)
  • Math.pow(x, y) → Returns x raised to the power y (Math.pow(2, 3) = 8)
  • Math.floor(x) → Rounds down to the nearest integer (Math.floor(4.7) = 4)
  • Math.ceil(x) → Rounds up to the nearest integer (Math.ceil(4.3) = 5)
  • Math.round(x) → Rounds to the nearest integer (Math.round(4.5) = 5)
  • Math.random() → Returns a random value between 0 and 1 (Example: 0.321654)

Infinity in JavaScript

`Infinity` is a special value that results when a number becomes mathematically infinite. It can be either positive or negative.

  • Infinity → Represents positive infinity
  • -Infinity → Represents negative infinity

Examples:

8 / 0 = Infinity

-8 / 0 = -Infinity

NaN in JavaScript

`NaN` stands for "Not a Number." It represents the result of an operation that does not produce a valid number.

  • 0 / 0 → NaN (Invalid division)
  • 'hello' * 2 → NaN (Multiplying a string with a number)

Examples:

0 / 0 = NaN

'hello' * 2 = NaN

document.writeln() in JavaScript
