String Methods and Properties

Strings are indexed, which means we can access any individual character of a string using its index number.

Expression: 'Hello World'[1]

Result: 'e'

Property

  • length → Returns the total number of characters in a string.

Methods (Without Argument)

  • toUpperCase() → Converts all characters of the string to uppercase.
  • toLowerCase() → Converts all characters of the string to lowercase.
  • trim() → Removes extra spaces from the beginning and end of the string.
  • trimStart() → Removes extra spaces from the beginning of the string.
  • trimEnd() → Removes extra spaces from the end of the string.

Methods (With Argument)

  • includes() → Checks if a substring exists in the string.
  • indexOf() → Returns the index of the first occurrence of a substring.
  • replace() → Replaces the first matching substring with the provided value.
  • replaceAll() → Replaces all matching substrings with the provided value.
  • concat() → Joins two or more strings.
  • padStart() → Pads the string from the beginning with a specific character until it reaches the desired length.
  • padEnd() → Pads the string from the end with a specific character until it reaches the desired length.
  • charAt() → Returns the character at a specific index.
  • charCodeAt() → Returns the ASCII value of the character at a specific index.
  • split() → Splits a string into an array of substrings based on a specified separator.

String Template Literal

Template literals allow you to embed expressions inside string literals using backticks (`) instead of quotes.

  • Use the ${expression} syntax to embed expressions in the string.
  • Example: const name = 'Arashad';
    const greeting = `Hello, ${name}!`;
  • Result: Hello, Arshad!
