What is an Operator?

An operator is a symbol that tells the compiler or interpreter to perform a specific mathematical, relational, or logical operation and produce a final result. For example, the addition operator + adds two values together.

Mathematical Operators

  1. Addition Operator: + → Adds two values (2 + 3 = 5)
  2. Subtraction Operator: - → Subtracts one value from another (5 - 3 = 2)
  3. Multiplication Operator: * → Multiplies two values (2 * 3 = 6)
  4. Division Operator: / → Divides one value by another (6 / 3 = 2)
  5. Remainder Operator: % → Returns the remainder after division (7 % 3 = 1)
  6. Exponentiation Operator: ** → Raises a value to the power of another (2 ** 3 = 8)
