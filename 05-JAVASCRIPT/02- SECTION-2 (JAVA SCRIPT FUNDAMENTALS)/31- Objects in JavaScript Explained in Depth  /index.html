JavaScript Data Types

There are 2 main types of data types in JavaScript:

  1. Primitive (Value Types) – Stores single values directly.
  2. Non-Primitive (Reference Types) – Stores a reference (memory address) to the value, not the value itself.

Primitive Data Types (7 types):

  1. Number – let age = 25;
  2. String – let name = "Arshad";
  3. Boolean – let isOnline = true;
  4. Undefined – let x;
  5. Null – let data = null;
  6. BigInt – let big = 12345678901234567890n;
  7. Symbol – let id = Symbol("id");

Non-Primitive Data Type: Object

Non-primitive types like objects don’t store the value directly. They store the reference (address in memory) where the value is located.

Simple Example:


let person1 = { name: "Arshad" };
let person2 = person1;

person2.name = "Ahmed";

console.log(person1.name); // Output: "Ahmed"

Explanation: Both person1 and person2 point to the same object in memory. Changing one affects the other.

Objects come in 3 main forms:

  • Object Literals – { key: "value" }
  • Arrays – [1, 2, 3]
  • Functions – function greet() { }
