Logical Operators in JavaScript: Numbers, Strings & Truthy/Falsy Values

Logical Operators:

Logical operators are symbols that help check multiple conditions in JavaScript. They either give true/false or check if a value is truthy/falsy.

AND (&&) Operator:

Rule: Returns the first falsy value; if all are truthy, returns the last one.

Example: 5 && 0 && 10 → Result: 0

Example: 7 && 9 && 15 → Result: 15

OR (||) Operator:

Rule: Returns the first truthy value; if all are falsy, returns the last one.

Example: 0 || "" || 10 || 20 → Result: 10

Example: 0 || "" || null || undefined → Result: undefined

NOT (!) Operator:

Rule: Converts truthy to false, falsy to true.

Example: !5 → Result: false

Example: !0 → Result: true

Truthy & Falsy Values in JavaScript:

Falsy Values: 0, "", null, undefined, NaN, false

Truthy Values: Everything else (like numbers 5, -10, "Hello")

Understanding console.log() Behavior:

If you type console.log('Hello'); in the browser's developer console (Chrome, Firefox, etc.), you will see "Hello" printed, but also "undefined". This happens because:

1️ console.log() does not return anything:

In JavaScript, console.log() is a function that prints a value to the console but does not return any value.

When a function does not return anything, JavaScript automatically returns undefined.

2️ Browser Console Behavior:

When you type a statement directly in the browser console and press Enter, the console shows the returned value of that statement.

Since console.log('Hello'); does not return anything, the console shows undefined.

Example 1: Using console.log()

console.log('Hello');

Output in Console:

Hello
undefined  // Because console.log() does not return anything
