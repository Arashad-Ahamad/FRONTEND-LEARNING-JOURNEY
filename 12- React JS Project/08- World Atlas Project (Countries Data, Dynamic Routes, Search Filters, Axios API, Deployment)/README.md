# 🌍 WorldAtlas - Country Explorer

A comprehensive React application that allows users to explore countries from around the world, view detailed information, search, filter, and sort through country data.

## 📄 Pages
- **Home** - Hero section with introduction
- **About** - Interesting country facts
- **Country** - Browse, search, filter, sort all countries
- **Country Details** - Detailed information of specific country
- **Contact** - Contact form for user queries
- **Error Page** - Custom error handling

## 🚀 Features
- **Browse Countries:** View a complete list of countries with essential information
- **Search Functionality:** Search countries by name in real-time
- **Filter by Region:** Filter countries by continents (Africa, Americas, Asia, Europe, Oceania)
- **Sort Countries:** Sort countries alphabetically in ascending or descending order
- **Country Details:** View detailed information about each country including:
  - Official name and native names
  - Population statistics
  - Region and subregion
  - Capital city
  - Top Level Domain
  - Currencies
  - Languages spoken
- **Responsive Design:** Mobile-friendly navigation with hamburger menu
- **Loading States:** Smooth loading transitions using React's useTransition
- **Error Handling:** Custom error page for routing issues

# 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| **React 18** | Frontend framework |
| **React Router DOM v6** | Routing and navigation |
| **Axios** | API calls |
| **React Icons** | Icon library |
| **Custom CSS** | Styling with media queries |
| **REST Countries API** | Country data source |
