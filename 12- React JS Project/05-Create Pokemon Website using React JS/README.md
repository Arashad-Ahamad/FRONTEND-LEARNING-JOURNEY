# React Pokémon App

A dynamic Pokémon web app built using React and PokéAPI.

## Features
- useState & useEffect hooks
- API fetching with async/await
- Promise.all for parallel requests
- Search functionality
- Filtering Pokémon
- Dynamic rendering
- Component-based architecture
- Responsive UI

## Tech Used
- React JS
- JavaScript
- CSS
- PokéAPI
